# Screen 01 — Splash / Landing

## Goal
Create the opening screen for MindMirror. This is the first impression of the app, so it must feel emotional, premium, and memorable.

## Required layout
- iPhone-style full-screen mobile layout.
- Dark background with a soft black-to-deep-navy gradient.
- Subtle purple glow or ambient light in the center/lower area.
- A cinematic illustration near the bottom: purple mountains, lake reflection, and a small seated human silhouette.
- App title placed prominently in the upper-middle or center area: “MindMirror”.
- A short subtitle: “AI Journaling that understands you.”
- Supporting message below:
  - “Reflect. Understand. Grow.”
  - Small text: “Your thoughts hold the answers. We help you find them.”
- Large primary CTA button near the bottom: “Start Your Journey”.
- Smaller secondary text button beneath it: “I already have an account”.

## Visual style
- Minimal, elegant, futuristic.
- Purple gradient for the app name and CTA button.
- Soft glow, slight blur, premium launch-screen mood.
- Rounded button corners.
- Use clean typography with strong contrast.

## Important details
- Do not overcrowd the screen.
- Keep the illustration emotionally calm, not noisy.
- The title should be highly readable and iconic.
- This screen should feel like a premium app intro, not a generic login page.
