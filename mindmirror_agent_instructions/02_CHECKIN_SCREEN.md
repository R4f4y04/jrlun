# Screen 02 — Daily Mood Check-in / Journal Entry

## Goal
This is the screen where the user logs how they feel and writes a journal entry for the day.

## Required layout
- Top greeting: “Good Morning, Amaan ✨”
- Follow-up prompt: “How do you feel today?”
- Mood selector row with five emoji faces:
  - Very Bad
  - Bad
  - Okay
  - Good
  - Great
- The selected mood must have a purple glow or highlighted state.
- Large text input card titled: “What’s on your mind?”
- Placeholder text:
  - “Write about your day, your thoughts, how you feel…”
- Two action buttons under the text box:
  - “Add Photo” with an image icon
  - “Add Voice” with a microphone icon
- Large primary gradient button at the bottom: “Save Entry”
- Bottom navigation bar with:
  - Home
  - Journal
  - Insights
  - Profile
- Journal should be active on this page.

## Visual style
- Dark mode with soft shadows.
- Rounded cards and buttons.
- Purple-blue gradient highlights.
- Clean spacing and calm emotional tone.

## Important details
- The page should feel easy and fast to use.
- The mood selector must be visually obvious.
- The text input should be the main focus.
- Keep the UI friendly and non-intimidating.
