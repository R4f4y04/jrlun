# Screen 05 — Journal Archive / Calendar

## Goal
Let the user browse past journal entries through a monthly calendar and a chronological list.

## Required layout
- Header with a back arrow.
- Month title: “May 2024”
- Small month navigation arrows if needed.
- Horizontal mini calendar showing a week view with dates.
- Selected date should be highlighted in purple.
- Below the calendar, show journal list cards for entries such as:
  - Today
  - Yesterday
  - 22 May
  - 21 May
- Each list item should include:
  - date/time
  - short preview of the journal entry
  - mood label on the right with emoji
- A floating circular purple plus button at the bottom center for creating a new entry.
- Bottom nav with Journal active.

## Visual style
- Dark list-based screen.
- Clean card rows with good spacing.
- Calendar should be compact but easy to read.
- Make the selected date visually distinct.

## Important details
- The list should feel like a memory timeline.
- Each item should be scannable in one glance.
- Keep the archive clean and structured.
