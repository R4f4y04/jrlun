# Screen 07 — Mood Trends Analytics

## Goal
Show mood behavior over time using charts and brief interpretation.

## Required layout
- Page title: “Mood Trends”
- Filter pill on top right: “This Week”
- Main line chart with days Mon to Sun.
- Mood scale labels should appear on the vertical axis:
  - Great
  - Good
  - Okay
  - Bad
  - Very Bad
- Use a glowing line with visible data points.
- Below the line chart, include a “Mood Distribution” donut chart.
- Legend should list mood categories and percentages.
- Include a “Best Day” card below with:
  - “Friday”
  - “You were most productive and positive.”
- Add a small trophy icon or illustration near the best-day card.

## Visual style
- Dark analytical dashboard.
- Clean charts with subtle grid lines.
- Teal or turquoise chart line.
- Smooth premium styling.

## Important details
- Make the chart easy to read on a phone.
- Keep the graphics uncluttered.
- The best-day card should feel like the key takeaway.
