# Screen 12 — Profile / Settings

## Goal
Show the user profile, usage stats, and core settings options.

## Required layout
- Top back arrow.
- Circular avatar.
- User name: “Amaan”
- Subtitle: “Growing every day 🌱”
- Three stats cards in one row:
  - Journals — 28
  - Streak — 12 days
  - Insights — 46
- Settings list with rounded rows:
  - Settings
  - Reminders
  - Export Data
  - Premium
- The Premium row should include a standout button on the right:
  - “Go Premium”
- Bottom navigation bar with Profile active.

## Visual style
- Clean dark profile page.
- Minimal and polished.
- Stats should be prominent.
- Premium CTA should feel special but not intrusive.

## Important details
- This screen should feel like the user’s home base.
- Keep the information hierarchy simple.
- The settings list must be easy to tap and visually consistent.
